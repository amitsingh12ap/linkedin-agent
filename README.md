# 🤖 LinkedIn Daily Post Agent

> Automatically posts one leadership-focused LinkedIn update every day at **9:00 AM IST** — generated by Claude AI in your voice. Runs entirely on **GitHub Actions** (free, no server needed).

---

## How It Works

```
GitHub Actions cron (9:00 AM IST daily)
        ↓
Reads data/agent.json  ←  theme rotation state
        ↓
Claude generates post in your voice
        ↓
LinkedIn API publishes it
        ↓
Commits updated history back to repo
```

## Setup

### 1. Fork / Clone this repo

```bash
git clone https://github.com/amitsingh12ap/linkedin-agent.git
cd linkedin-agent
npm install
```

### 2. Add GitHub Secrets

Go to **Settings → Secrets and variables → Actions → New repository secret** and add:

| Secret | Where to get it |
|--------|----------------|
| `ANTHROPIC_API_KEY` | [console.anthropic.com](https://console.anthropic.com) → API Keys |
| `LINKEDIN_ACCESS_TOKEN` | LinkedIn OAuth — see below |
| `LINKEDIN_PERSON_URN` | Run `npm run get-urn` after setting the token |

### 3. Get LinkedIn Credentials

**Step 1** — Create a LinkedIn App at [linkedin.com/developers/apps](https://www.linkedin.com/developers/apps)
- Request products: **"Share on LinkedIn"** + **"Sign In with LinkedIn using OpenID Connect"**

**Step 2** — Get an OAuth token using the [LinkedIn OAuth Playground](https://www.linkedin.com/developers/tools/oauth)
- Required scopes: `openid profile w_member_social`

**Step 3** — Get your Person URN:
```bash
LINKEDIN_ACCESS_TOKEN=your_token npm run get-urn
# → urn:li:person:ABC123xyz
```

> ⚠️ LinkedIn tokens expire every **60 days**. Set a calendar reminder to refresh.

## Running

### Automatic (GitHub Actions)
The workflow triggers automatically at **3:30 AM UTC (9:00 AM IST)** every day.

### Manual trigger
Go to **Actions → Daily LinkedIn Post → Run workflow** — you can also override the theme.

### Local testing
```bash
cp .env.example .env    # fill in your keys
npm run preview         # generate a post without publishing
npm run post-now        # publish one post immediately
npm run history         # view past posts
```

```
│   └── agent.json          # Theme rotation state + history (auto-committed)

```

## Customising Your Voice

Edit `src/generator.js` → `VOICE_PROFILE` to tune the writing style. For best results, paste 3–5 of your best past LinkedIn posts directly into the profile as examples.

| Command | Description |
